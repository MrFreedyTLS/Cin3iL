#Description générale
Ce projet consiste en un site web pour un cinéma, qui permet de visualiser les films actuellement à l'affiche et tous les films proposés par le cinéma, y compris ceux qui ne sont pas en cours de projection.

#Fonctionnalités pour les utilisateurs
En tant qu'utilisateur du site, vous avez accès à plusieurs fonctionnalités :
	- Consulter les films à l'affiche : Une liste des films actuellement projetés au cinéma.
	- Explorer tous les films proposés : Une vue complète de tous les films disponibles, même ceux qui ne sont pas actuellement à l'affiche.
	- Accéder à la galerie photos : Une section galerie vous permet de consulter des photos du cinéma.
	- Informations pratiques : Vous pouvez obtenir des informations essentielles comme les horaires d'ouverture, l'adresse, le lieu et les moyens de contact du cinéma.
	- Formulaire de contact : Un formulaire intégré permet aux utilisateurs de contacter directement le cinéma pour toute question ou demande.

#Fonctionnalités pour l'administrateur
Le site propose un espace administrateur permettant de gérer les films à l'affiche :
	- Connexion administrateur : Un administrateur peut se connecter pour accéder à l'interface de gestion.
	- Gestion des films : Une fois connecté, l'administrateur peut ajouter, modifier ou supprimer des films à l'affiche.
	- Création de compte administrateur : Il est possible de créer un compte administrateur en cliquant sur le bouton "S'inscrire". Si besoin, vous pouvez utiliser les informations de connexion suivantes :
		- Login : admin
		- Password : admin

#Base de données
Le dossier contient également un script SQL permettant d'initialiser la base de données du site.
